# How to use the Immutable X API to get asset metadata

[API reference](https://docs.x.immutable.com/reference)

## Developer tools

Immutable provide a number of tools for your application to register users:

| Tool                                     | Language                  | Steps covered (from [here](#what-is-user-registration)) | Description |
| ---------------------------------------- | ------------------------- | ------------- | ----------- |
| **[Link SDK](https://www.npmjs.com/package/@imtbl/imx-sdk)** | Typescript | 1, 2 | Provides end-to-end user wallet connection and registration, complete with a UI |
| **Wallet SDK** | [Web](https://github.com/immutable/imx-wallet-sdk-web), [Android](https://github.com/immutable/imx-wallet-sdk-android) | 1 | Provides a connection to users' L1 and L2 wallets and returns 'signers' that can be used to sign transactions. No UI. |
| **Core SDK** | [Typescript](/docs/sdk-docs/wallet-sdk-web/overview.md), [Kotlin (JVM)](/docs/sdk-docs/core-sdk-kotlin/overview.md), [Swift](/docs/sdk-docs/core-sdk-swift/overview.md) | 2 | Performs off-chain registration of users. Requires 'signers' to obtain user signatures - can be provided by the **Wallet SDK** or implemented manually |
| **[API](https://docs.x.immutable.com/reference#/)** | -- | 1, 2 | Provides endpoints for connection to user wallets, obtaining signatures and registering users off-chain |
