A user can transfer the ownership of an asset that they own to another user.

**Available with:**
* [Core SDK](#core-sdk)
* [StarkNet](#starknet)
* [Link SDK](/docs/linktransfer)

## Core SDK

### 1. Initialize the Core SDK
In order to use the Core SDK, you need to [initialize it](./get-started#getting-started-with-the-core-sdk).

### 2. Generate signers
Transferring an asset requires a user's signature, so your application will need to [generate signers](/docs/guides-new/generate-signers).
